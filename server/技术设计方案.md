### 后端技术栈选型

| 模块         | 技术选型                                      |
|:-------------|:---------------------------------------------|
| 核心框架     | Koa2 + TypeScript                             |
| ORM          | Mongoose (MongoDB ODM)                       |
| 身份认证     | JWT + koa-jwt                               |
| 文件上传     | koa-body + koa-multer                        |
| 接口规范     | RESTful API + Swagger（可选）                |
| 请求验证     | Joi                                         |
| 错误处理     | Koa中间件统一处理                            |
